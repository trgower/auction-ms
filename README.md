# auction-ms
Auction site for an older version of MapleStory. Images not included.

# Features
+ Registration
+ Email Verification
+ Password reset
+ Password change
+ Searching and Pagination
+ Search autocomplete
+ Auctions
+ Item Stats
+ Bidding
+ Comment System
+ User Page
+ Private messaging
+ Honor system (feedback)
+ Sorts search results
+ Reserve price (replaced buyout)
+ Dashboard
+ Watch Items
+ Alerts
